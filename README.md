# Get Image
This code get images from file.
I verified several extensions below.
- [x] .docx
- [x] .pptx
- [ ] .xlsx
- [ ] .md

This code doesn't guarantee that images can be retrieved for all files.

## Get images extensions
- .png
- .jpg
- .jpeg

## Run
```
py getImages.py <targetFilePath>
```
